### 后端

### 前端

- `react` 基本 UI 框架库
- `react-dom` 渲染 UI
- `react-router` 路由及导航管理
- `vite` 构建系统
- 状态管理
    - `react-redux` 集成 Redux 到 React 中
    - `@reduxjs/toolkit` 简化 redux 使用

- UI
    - `antd` 企业级 UI 开发库
- 样式
    - `styled-components` 样式组件库
